## Setup Instructions

### Prerequisites
This project is built with modern React tooling. Make sure you have the following installed:

- Node.js (v18+ recommended)
- npm
- React 19
- React Router
- Tailwind CSS
- Ant Design (antd)

> React, React Router, Tailwind, and antd are installed as project dependencies. You do not need them globally.

---

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/Maobugichi/flowva-test.git
cd flowva-test
npm install
npm run dev
